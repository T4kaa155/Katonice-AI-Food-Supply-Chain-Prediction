# Prototype Aplikasi

Prototype aplikasi ini merupakan implementasi awal dari sistem AI Food Supply Chain berbasis data.

## 🔗 Demo Aplikasi
Link Google Drive:
[https://drive.google.com/your-link-here](https://drive.google.com/drive/folders/132olhbCdGFMHaLuzQS_EhyyDLmk9hXyx?usp=sharing)

## 🎯 Fitur Prototype
- Input data produksi & permintaan
- Prediksi harga menggunakan AI model
- Estimasi panen
- Analisis supply-demand gap
- Rekomendasi distribusi

## 💡 Tujuan
Prototype ini menunjukkan bagaimana model AI dapat digunakan dalam aplikasi nyata untuk:
- Monitoring kondisi pangan
- Membantu pengambilan keputusan
- Mengurangi food waste
