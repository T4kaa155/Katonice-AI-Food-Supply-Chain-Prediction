# Prototype Aplikasi

Prototype ini merupakan tampilan awal (visual design) dari sistem AI Food Supply Chain yang dikembangkan.

## Demo Prototype
Link Google Drive:
[https://drive.google.com/your-link-here](https://drive.google.com/drive/folders/132olhbCdGFMHaLuzQS_EhyyDLmk9hXyx?usp=sharing)

## Fitur Prototype (Visual)
- Tampilan input data (produksi, permintaan, lokasi)
- Dashboard monitoring supply chain
- Visualisasi data (produksi, harga, distribusi)
- Tampilan hasil prediksi (simulasi)

## Status Pengembangan
Prototype ini masih berupa visual (UI/UX design) dan belum terintegrasi langsung dengan model AI.

Namun, seluruh fitur yang ditampilkan telah disesuaikan dengan sistem AI yang dibangun pada notebook.

## Keterkaitan dengan AI System
- Model AI telah dibuat di bagian notebook
- Prototype ini menggambarkan bagaimana output AI akan digunakan dalam aplikasi
- Prediksi seperti harga, panen, dan distribusi ditampilkan dalam bentuk UI

## Tujuan
Prototype ini bertujuan untuk:
- Menunjukkan konsep implementasi AI dalam aplikasi nyata
- Memberikan gambaran alur penggunaan sistem
- Menjadi dasar pengembangan aplikasi ke tahap production

## Future Development
- Integrasi langsung dengan model AI (real-time prediction)
- Deployment berbasis cloud (Azure / API)
- Fitur monitoring & notifikasi otomatis
- Sistem rekomendasi berbasis AI
